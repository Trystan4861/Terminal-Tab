## Características

- Abre una terminal como una pestaña en el área del editor y ejecuta el comando configurado
- Comando configurable (valor por defecto: `whoami`)
- Acción en la paleta de comandos para cambiar el comando
- Acción en la paleta de comandos para abrir directamente la configuración del comando
- El tooltip de la barra de estado muestra el comando configurado actual
- Reutiliza la terminal existente si ya está abierta
- Atajo predeterminado: `Cmd+F1` (macOS) / `Ctrl+F1` (Windows/Linux)

## Uso

Pulsa `Cmd+F1` (macOS) o `Ctrl+F1` (Windows/Linux) para abrir una pestaña de terminal en el área del editor, o ejecuta el comando **Terminal Tab: Open Terminal** desde la paleta de comandos.
